Shifter’s Component guide class.

class mgear.maya.shifter.component.guide.ComponentGuide¶

Main class for component guide creation. This class handles all the parameters and objectDefs creation. It also know how to parse its own hierachy of object to retrieve position and transform. Finally it also now how to export itself as xml_node.
